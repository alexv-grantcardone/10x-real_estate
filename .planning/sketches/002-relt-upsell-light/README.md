# Sketch 002: RELT $97 Upsell — Light-Dominant

> **Note:** Different funnel from sketch 001 (which is RE Summit). This is the
> **RELT** funnel: opt-in (the live-training "Control") → VIP checkout ($27) →
> **this $97 one-click upsell** → access page. Palette/fonts stay in the RELT
> family (navy `#0A2540`, teal `#13DDE2`, Inter + Bodoni-italic); only the
> light/dark balance flips.

## Design Question
The first build was navy-dominant (dark background). This explores flipping it:
**light background dominant, navy + teal as the darker accents.** Same content,
same tokens, same fonts — different visual weight.

## How to View
`open .planning/sketches/002-relt-upsell-light/index.html`
(Use the top tab bar to switch A / B / C. Resize the real browser window for
true mobile — the toolbar "Phone" button only narrows the container.)

## Variants
- **A · Clean Light Card** — white centered card on a light page, compact and conversion-first. Navy = text, teal = CTA + checks. Closest to a premium SaaS checkout.
- **B · Navy Anchor Block — ★ WINNER** — light page with the price + CTA in a sticky **navy accent card** (inverts the old layout so navy now anchors the action). Two-column on desktop; stacks on mobile with a **sticky bottom buy bar**.
- **C · Editorial Airy** — large Bodoni display, numbered hairline list, generous white space, image-forward split, full-width price/CTA band. Navy = serif accents; teal reserved for the button.

## Winner: B — refinements applied (2026-06-25)
- Headline → **"Before You Go — Gain Access to the Master Class Bundle"**
- Real product image dropped in, **red "REAL ESTATE MASTERCLASS BUNDLE" banner cropped out** (off-brand red + repetitive wording); clean two-monitor product shot on a light surface with soft shadow.
- Removed the repetitive "The Real Estate Master Class Bundle" title that sat under the image (headline now names it once; anchor card names it once).
- **Mobile-optimized:** two-column → stacked (image → bullets → navy price/CTA anchor), plus a **sticky bottom $97 buy bar** so the CTA is always reachable.
- Cleaned image asset saved: `mc-bundle-clean.webp` / `.png` (scratchpad) — needs uploading to GHL, OR crop the original URL via CSS in the paste block.

## What to Look For
- Which light treatment still feels like Grant Cardone / RELT (premium, confident) vs. generic?
- Does navy-as-accent read as intentional, or does the page lose its anchor without the dark ground?
- CTA prominence on light — is the teal button strong enough without the dark backdrop?
- Cohesion: would this sit naturally after the (still navy-dominant) opt-in + VIP pages, or does flipping the upsell break the funnel's continuity?
