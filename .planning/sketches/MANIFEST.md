# Sketch Manifest

## Design Direction
**Working palette (chosen 2026-06-24): LIGHT / AIRY** — light blue-gray page (`#EEF3F8`), white/tinted surfaces, navy text (`#06192c` headings / `#33475a` body), deepened cyan-blue accent (`#0E97C7`, readable on white). Clean and trustworthy; the official ticket art pops as the color focal point. The original **dark-navy + cyan** palette is preserved as the `navy.css` alternate (switchable via the sketch toolbar). Typography: Inter body with a Bodoni Moda **upright** display accent (never italic). Hard copy rules: **no italics**, and the word **"FREE" is banned** (use "no payment today" / "held 10 minutes"). Editorial, confident, Grant Cardone real-estate brand; look-and-feel adapted from a RELT reference block (aesthetics only). Token-driven — the whole system re-skins from `themes/`. NOTE: Paige's official brand palette is still pending; this is a strong working direction, not a final lock. Applies to the RE Summit "RSVP MY SEAT" funnel.

## Reference Points

- Current live opt-in page content (Grant's story, stats, tier inclusions)
