# Sketch Manifest

## Design Direction
Premium dark-navy (`#0A2540`) + bright cyan accent (`#13DDE2`) + bone (`#EAEAEA`), Inter body with a Bodoni Moda **upright** display accent (never italic — italics are banned for readability). Editorial, confident, Grant Cardone real-estate brand. Hard copy rules: **no italics**, and the word **"FREE" is banned** (use "no payment today" / "held 72 hours"). Look-and-feel adapted from a Real Estate Live Training reference block (aesthetics only). Applies to the RE Summit "RSVP MY SEAT" funnel.

## Reference Points
- RELT reference code block (navy/cyan, serif accent, marquee, stat band) — aesthetics only
- Current live opt-in page content (Grant's story, stats, tier inclusions)
- Marketing survey VOC (capital is the #1 blocker; they want finding deals + raising capital + full process)
